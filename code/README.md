# NLP 课程配套最小代码（可直接运行）

运行方式（在仓库根目录）：

```bash
python code/vit_patchify.py
python code/clip_contrastive.py
python code/lora_linear.py
python code/reward_model_bradley_terry.py
python code/ppo_skeleton.py
python code/huffman_tree.py
python code/hierarchical_softmax.py
python code/rnn_language_model.py
python code/data_quality_dedup.py
python code/glu_swiglu_plot.py
```

说明：
- 这些脚本尽量只依赖 `numpy`（已在 `requirements.txt` 里）；`glu_swiglu_plot.py` 有 `matplotlib` 则画图，否则落 CSV。
